# Azure OpenAI — Overview (Responses API)

- **Primary interface:** `/openai/v1/responses` on your Azure resource; supports stateful turns, tools, structured outputs, and streaming.
- **Models & regions:** See the how‑to for currently supported regions and model families (includes GPT‑5 series, o‑family, gpt‑4.1, 4o, image models).
- **Not currently supported (notable):** Web search tool; multi‑turn image gen edits/streaming; image upload as file input (coming). Known issues and notes are tracked on the page.
