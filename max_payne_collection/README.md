# Full Collection - Payne Amplifier

Complete collection of all payne amplifier components including:

- Issue-aware development profile
- Persistent issue tracking with dependency management
- Custom tools and modules for enhanced productivity

## Installation

```bash
amplifier collection add "git+https://github.com/payneio/amplifier-payne@main#subdirectory=max_payne_collection"
```

### Profiles
- Issue-aware development workflows
- Custom agent configurations

### Modules
- `tool-issue`: Issue tracking tool integration
- `issue-manager`: Persistent issue queue management

### Context
